# AI Prompt Guidelines (작업 지침서, 작업 지시서)

## 최우선 사항
- 모든 AI 질문, 응답, 주석 작업, 커밋 로그 등 되도록 한글을 사용한다
- 이 파일은 AI 에이전트(Gemini CLI 등)가 이 프로젝트를 수행할 때 참고해야 하는 전역 지침과 프로젝트 정보를 담고 있습니다.

## 프로젝트 개요
- **이름**: gitnote
- **주요 목적**: 깃허브를 저장소로 사용하는 메모장 앱

## 작업 일지(journal.md) 작성 규칙
- 날짜 시간, 내용을 포함
- 각 저널 뒤에 `---` 표시로 줄 바꿈과 엔터(공백 줄바꿈을 추가)
- 작업 일지 과거 내용은 삭제하지 않고, 작업 내용을 누적 시킨다

## AI 지침
- AI 가 작업하는 내용은 .ai 폴더 하위에 기록, 참고 한다
- AI 가 생각을 하기 전에 .ai/prompts 하위 문서를 먼저 읽고 생각을 시작한다
  - 필요한 경우 md 파일을 수정한다
- AI 의 작업 결과물 중 저널, 결과 리포트 등은 .ai/llm_outs 폴더 하위에 기록한다
- PROMPT.md 파일은 AI 가 수정하지 않는다
- 코드 수정, 작성은 되도록 최소한의 코드를 사용한다
- 작업 목록은 미진행([ ]), 완료([x]), 진행중([/]) 표기를 사용하여 목록으로 관리한다
- 작업을 1개 완료하면 작업 일지를 반드시 작성한다
- 작업 1개마다 검증 작업 1개를 되도록이면 같이 진행한다
- 작업 내용에 **'스크린샷 참고'** 가 있을 경우, .ai/assets/bugs 폴더를 우선적으로 검색해서 스크린샷 이미지를 참고 한다
